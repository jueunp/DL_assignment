# Foundation Models Midterm Project

This repository contains three small demos built with Hugging Face foundation models for the midterm assignment:

1. Point-based segmentation
2. Open-vocabulary detection
3. Vision-language chat

The goal of this project is to compare how different vision foundation models behave on real images rather than only reading about them.

## Repository Structure

```text
foundation_models/
  open_vocab_detection.py
  point_based_segmentation.py
  vision_language_chat.py
utils/
  image_io.py
requirements.txt
README.md
```

## Installation Guide

### 1. Create a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 2. Install dependencies

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 3. Notes

- A GPU is recommended, but all scripts can fall back to CPU.
- The first run may take time because model weights are downloaded from Hugging Face.
- If you run on Apple Silicon, `mps` may also work depending on your PyTorch installation.

## Demo 1: Open-Vocabulary Detection

File: [foundation_models/open_vocab_detection.py](/Users/cookie/Documents/New project/foundation_models/open_vocab_detection.py)

### Model

- `IDEA-Research/grounding-dino-tiny`

### What it does

Given an image and text labels such as `"person, phone, clock"`, the model finds matching objects and draws bounding boxes.

### Input

- Image path or URL
- Comma-separated text prompts
- Optional confidence thresholds

### Output

- Saved image with labeled bounding boxes
- Console summary with detected objects and scores

### Example

```bash
python foundation_models/open_vocab_detection.py \
  --image path/to/photo.jpg \
  --labels "person, clock, laptop" \
  --output docs/open_vocab_result.png
```

### Example Result

- The model draws red boxes around objects matching the requested labels.
- Example detections might look like:
  - `person: 0.92`
  - `clock: 0.81`

## Demo 2: Point-Based Segmentation

File: [foundation_models/point_based_segmentation.py](/Users/cookie/Documents/New project/foundation_models/point_based_segmentation.py)

### Model

- `facebook/sam2-hiera-small`

### What it does

Given an image and one or more user-provided points, the model predicts a segmentation mask for the object indicated by the point prompts.

### Input

- Image path or URL
- Point coordinates such as `x,y`
- Optional point labels (`1` for foreground, `0` for background)

### Output

- Saved visualization with point prompts and mask overlay
- Console summary with mask confidence scores

### Example

```bash
python foundation_models/point_based_segmentation.py \
  --image path/to/photo.jpg \
  --points "320,240" \
  --output docs/segmentation_result.png
```

### Example Result

- The selected object region is highlighted with a translucent mask.
- A foreground point on a person usually produces a person-shaped mask.

## Demo 3: Vision-Language Chat

File: [foundation_models/vision_language_chat.py](/Users/cookie/Documents/New project/foundation_models/vision_language_chat.py)

### Model

- `HuggingFaceTB/SmolVLM-Instruct`

### What it does

This script loads an image and asks a visual question, such as describing the scene or identifying important objects.

### Input

- Image path or URL
- Natural-language question
- Optional maximum number of generated tokens

### Output

- Text answer generated by the vision-language model

### Example

```bash
python foundation_models/vision_language_chat.py \
  --image path/to/photo.jpg \
  --question "What is happening in this image?"
```

### Example Result

- Example answer:
  - `"The image shows a person standing indoors near a wall clock and holding an object that looks like a phone."`

## Brief Explanation of Each Code

### `utils/image_io.py`

- Loads images from a local path or URL
- Converts images to RGB
- Selects an available device (`cuda`, `mps`, or `cpu`)

### `foundation_models/open_vocab_detection.py`

- Loads Grounding DINO processor and model
- Runs zero-shot object detection using user text prompts
- Saves a visualization with boxes and labels

### `foundation_models/point_based_segmentation.py`

- Loads SAM2 processor and model
- Accepts point prompts from the user
- Predicts masks and saves an overlay image

### `foundation_models/vision_language_chat.py`

- Loads a compact vision-language instruction model
- Builds a multimodal prompt with an image and user question
- Prints the generated answer to the terminal

## Reproducibility

All scripts are standalone and can be run from the command line. To reproduce results:

1. Install dependencies
2. Prepare an image
3. Run any of the three scripts with your own inputs
